front-end

install react-router-dom to make routes
    yarn add react-router-dom

install axios to make http requests
    yarn add axios


Back-end

install express
    yarn add express

install cors (to allow front-end to access back-end there is two different ports that's why we need cors)
    yarn add cors

install mongoose (to connect to mongodb)
    yarn add mongoose

install nodemon (to restart server automatically)
    yarn add nodemon -D

install dotenv (to hide sensitive information)
    yarn add dotenv

install bcrypt (to encrypt password)
    yarn add bcrypt
